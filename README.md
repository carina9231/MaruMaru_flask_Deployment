## 📌 주요 기능

1. 카카오 API 를 활용한 지도 모달창
2. 지도 카데고리별 커스텀 검색 결과 제공
3. ajax를 활용한 게시글 CRUD 기능
4. Ajax와 mongoDB 기능을 활용한 비동기 댓글 기능
5. JWT를 활용한 회원가입, 로그인 기능
6. 로그인한 유저만 게시물 & 이벤트 생성, 수정, 삭제 가능
7. 이벤트에 참여한 유저 정보 확인 가능
8. 마음에 드는 이벤트, 게시물에 좋아요 가능
<br/>
